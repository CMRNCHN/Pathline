# Pathline AI Agent Rules

Planning artifacts are not implementation authorization.

## Forbidden Changes
- websocket rewrites
- replay semantic changes
- topology changes
- framework migrations
- hidden orchestration
- autonomous traversal logic
- build-system additions
- frontend framework injection

## Protected Systems
- replay chronology
- deterministic traversal
- inspection diagnostics
- route verification
- checkpoint semantics
- bounded execution

## Frontend Constraints
- vanilla JS only
- no hidden state frameworks
- no React/Vue/Svelte
- no build pipeline

## Runtime Constraints
- preserve deterministic execution
- preserve replay semantics
- preserve protocol boundaries
- preserve topology

## Commit Discipline
Separate:
- governance/docs
- frontend UX
- runtime/backend
- replay semantics
- storage changes
